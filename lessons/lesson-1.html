Урок 1 Введение и установка Bootstrap

Что такое Bootstrap

Bootstrap - это фреймворк для создания веб-приложений и сайтов. Он позволяет быстро и легко создавать адаптивные интерфейсы, поддерживает различные устройства и браузеры. Bootstrap включает в себя набор готовых компонентов, таких как кнопки, формы, навигационные панели, таблицы и другие, которые можно использовать для создания своего проекта. Фреймворк также предоставляет инструменты для стилизации и кастомизации элементов интерфейса.

Основная цель Bootstrap - упростить процесс разработки веб-проектов, сократить время на создание адаптивных интерфейсов и повысить качество кода. Он подходит как для начинающих разработчиков, так и для опытных программистов. Bootstrap активно развивается и обновляется, что гарантирует поддержку современных технологий и стандартов.

Преимущества Bootstrap:

  • Быстрая разработка: благодаря готовым компонентам и шаблонам, можно значительно ускорить процесс создания веб-проектов.
  • Кросс-браузерная совместимость: Bootstrap обеспечивает корректное отображение веб-интерфейса во всех популярных браузерах.
  • Отзывчивость и адаптивность: Bootstrap автоматически адаптирует элементы веб-страницы под разные разрешения экрана, делая их доступными для просмотра на различных устройствах.

Установка Bootstrap

Для начала нам необходим обычный html файл. Для того чтобы пользоваться bootstrap'ом, нужно его сначала подключить к нашему html файлу. Сделать это можно тремя основными способами:

  • Загрузка Bootstrap с официального сайта и добавление его вручную в проект
  • Подключение Bootstrap через CDN (Content Delivery Network)
  • Использование Bootstrap через npm или yarn

Стоит уточнить, что на момент разработки данного учебного пособия, актуальная версия bootstrap - 5.3.2, Но вы всегда можете найти новейшую версию в ссылке, что находится в шапке сайта. Там же вы найдёте и документацию.

Подробная инструкция по подключению. Первый способ:

1. Перейдите на официальный сайт Bootstrap и скачайте последнюю версию (обычно это архив в формате .zip). Ссылка на скачивание находится на главной странице сайта фреймворка (Выделена красным на изображении ниже).
2. Распакуйте архив и скопируйте из него папки css и js в корень вашего проекта.
3. Подключите с помощью link к вашему html файлу соответствующие bootstrap'у файлы (я рекомендую подключать bootstrap.min).

В итоге в нашем html файле должен быть следующий код: 

<!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <link rel="stylesheet" href="css/bootstrap.min.css">
        <title>Document&lt/title>
    </head>
    <body>
        <script src="js/bootstrap.min.js">&lt/script>
    </body>
</html>

Второй способ подключения:

Чтобы подключить Bootstrap через CDN, добавьте следующий код в тег head вашей HTML-страницы (Актуальную ссылку на новейшую версию bootstrap так же можно найти на главной странице сайта фреймворка): 

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-T3c6CoIi6uLrA9TneNEoa7RxnatzjcDSCmG1MXxSR1GAsXEV/Dwwykc2MPK8M2HN" crossorigin="anonymous">

Третий способ подключения:

Третий способ - это загрузить bootstrap при помощи npm, и подключить его точно так же, как в 1 способе.
Для этого нам понадобится выполнить в командной строке следующую команду: 

npm i install bootstrap@v5.3.2

Теперь вы подключили фреймворк bootstrap к своему проекту, и можете переходить к уроку 2.
