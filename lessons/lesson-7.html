Урок 7 Компоненты Bootstrap. Навигация и пагинация

Навбар

Для создания навигационной панели необходимо создать контейнер с классом .nav, а так же требуется оболочка .navbar-expand{-sm|-md|-lg|-xl|-xxl} для адаптивного свертывания и классов цветовой схемы. По умолчанию навигационные панели и их содержимое являются гибкими. Измените контейнер, чтобы ограничить их горизонтальную ширину разными способами. Обеспечьте доступность с помощью <nav> элемента или, если используется более общий элемент, такой как <div>, добавьте role="navigation" к каждой панели навигации, чтобы явно обозначить ее как ориентирную область для пользователей вспомогательных технологий.

.navbar-nav для полноразмерной и облегченной навигации (включая поддержку раскрывающихся списков). .navbar-toggler для использования с нашим плагином свертывания и другими функциями переключения навигации.

Вот пример всех подкомпонентов, включенных в адаптивную навигационную панель, которая автоматически сворачивается в lg точке останова. 

<nav class="navbar navbar-expand-lg bg-body-tertiary">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">Navbar</a>
        <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarSupportedContent" aria-controls="navbarSupportedContent" aria-expanded="false" aria-label="Toggle navigation">
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse" id="navbarSupportedContent">
            <ul class="navbar-nav me-auto mb-2 mb-lg-0">
                <li class="nav-item">
                    <a class="nav-link active" aria-current="page" href="#">Home</a>
                </li>
                <li class="nav-item">
                    <a class="nav-link" href="#">Link</a>
                </li>
                <li class="nav-item">
                    <a class="nav-link disabled" aria-disabled="true">Disabled</a>
                </li>
            </ul>
            <form class="d-flex" role="search">
                <input class="form-control me-2" type="search" placeholder="Search" aria-label="Search">
                <button class="btn btn-outline-success" type="submit">Search</button>
            </form>
        </div>
    </div>
</nav>

Навигационные ссылки на панели навигации основаные на .nav опциях, имеют собственный класс модификаторов и требуют использования классов-переключателей для правильного адаптивного стиля. Навигация в навигационных панелях также будет занимать как можно больше горизонтального пространства, чтобы содержимое вашей навигационной панели было надежно выровнено. Добавьте класс .active в .nav-link чтобы обозначить текущую страницу. Обратите внимание, что вам также следует добавить aria-current атрибут чтобы активировать .nav-link.

По умолчанию используется светлая тема навигации, но вы также можете применить data-bs-theme="dark" к .navbar родительскому элементу темные цвета фона. 

<nav class="navbar bg-dark border-bottom border-body" data-bs-theme="dark">
    <!-- Navbar content -->
</nav>

<nav class="navbar bg-primary" data-bs-theme="dark">
    <!-- Navbar content -->
</nav>

<nav class="navbar" style="background-color: #e3f2fd;">
    <!-- Navbar content -->
</nav>

Если вы используете тег <nav> для предоставления панели навигации, обязательно добавьте элемент role="navigation" в наиболее логичный родительский контейнер <ul> или оберните <nav> элемент вокруг всей навигации.

Пагинация

Разбивка на страницы построена на основе HTML-элементов списка, поэтому программы чтения с экрана могут сообщать количество доступных ссылок. Используйте элемент-обертку <nav>, чтобы идентифицировать пагинацию как раздел навигации для программ чтения с экрана и других вспомогательных технологий.

Кроме того, поскольку на страницах, скорее всего, имеется более одного такого раздела навигации, целесообразно предусмотреть описательную метку aria-label для <nav>, отражающую его назначение. Например, если компонент пагинации используется для навигации между набором результатов поиска, подходящей меткой может быть aria-label="Страницы результатов поиска". 

<nav aria-label="Page navigation example" class="example" data-bs-theme="dark">
    <ul class="pagination">
        <li class="page-item"><a class="page-link" href="#">Previous</a></li>
        <li class="page-item"><a class="page-link" href="#">1</a></li>
        <li class="page-item"><a class="page-link" href="#">2</a></li>
        <li class="page-item"><a class="page-link" href="#">3</a></li>
        <li class="page-item"><a class="page-link" href="#">Next</a></li>
    </ul>
</nav>
